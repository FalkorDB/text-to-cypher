# Text-to-Cypher Hero GIF

This project renders the hero GIF used in the repository README.

## Commands

Install dependencies:

```console
npm i
```

Start preview:

```console
npm run dev
```

Update code snippets:

- Edit files in the `public` folder.
- The render order follows the `code*` filenames.

Render the GIF:

```console
npx remotion render Main out/hero.gif --codec=gif
```
